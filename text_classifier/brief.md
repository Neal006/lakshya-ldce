**Complaint Text Classifier** — FastAPI inference server that classifies customer complaints by category (Trade/Product/Packaging), sentiment (VADER score), and priority (DecisionTree). Uses an ensemble of DistilBERT-MNLI zero-shot classification and MiniLM-L6 semantic similarity via ONNX Runtime (CUDA-accelerated). Three endpoints: /health, /predict (JSON), /metrics (Prometheus). Pipeline: text -> ONNX zero-shot + similarity ensemble -> category, VADER -> sentiment, DecisionTree -> priority. Run: python run_server.py.
