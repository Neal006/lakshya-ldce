**Speech-to-Text Server** — FastAPI inference server for audio transcription using faster-whisper (CTranslate2 int8/int8_float16) with Silero VAD ONNX for silence removal. GPU-accelerated on CUDA, CPU fallback. Five endpoints: /health, /transcribe (file upload), /transcribe/raw (PCM16), /ws/transcribe (WebSocket streaming), /metrics (Prometheus). Pipeline: audio -> resample 16kHz -> VAD silence removal -> normalize -> faster-whisper -> postprocess (filler removal, number normalization, casing). Auto-downloads VAD model. Run: python run_server.py.
